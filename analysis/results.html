Where it was wrong: too humble

Calibration is the real test of a probability model — when it says 60%, does that happen 60% of the time? Pooling all 309 forecasts, one clear pattern emerged: the model's favourites won more often than it predicted. This World Cup had fewer upsets than the odds implied.

04

The honest null result

The one genuinely novel thing here was the context layer — nudging each match for stadium heat, rest days and travel distance. So: did it work?

No. On this tournament, the hand-tuned context layer did not improve the forecasts.

Brier change (worse)

Across the 80 matches where heat, rest or travel moved the numbers, stripping the whole context layer out changed accuracy by zero and nudged the Brier score a hair in the wrong direction. The adjustments were built from published research priors, not fit to data — and it shows. Reporting this is the point: the model kept a public track record precisely so a null like this couldn't be quietly buried.

05

Is it a paper?

The straight answer: not a traditional methods paper — but there's a genuine, publishable artifact inside this if it's framed for what it actually is.

✕ Not as novel methodology

  • Elo + Poisson + Monte Carlo is textbook — 40 years old, not a contribution
  • n = 103 is one tournament: far too small for statistical validation of a model
  • The context layer's headline result is a null — no measured edge to claim

✓ Yes as a pre-registered case study

  • Every forecast was timestamped and frozen before kickoff — a real audit trail, which is rare and the strongest scientific asset
  • Honest calibration + a reported null model exactly the integrity reviewers reward
  • Fits an applied venue (e.g. Sloan Sports Analytics) or a solid preprint / technical writeup
06

What's next

1

Fit the context layer instead of hand-setting it

The null is a launchpad, not a dead end. Fit heat/rest/travel coefficients on decades of historical internationals, then test them out-of-sample. Turning "we guessed the weights" into "we learned them, and here's whether they beat plain Elo" is the one move that makes this genuinely novel.

2

Write the case study around the audit trail

Lead with the pre-registration. The dataset — 810 frozen prediction snapshots, publicly logged, plus results — is the paper's spine. Publish it openly alongside the writeup.

3

Ship the retrospective on the live site

Turn the Bracket into a "how it finished vs how we called it" view, and freeze the Final Report Card as a permanent tab. The project earned a closing chapter, not a quiet sunset.
